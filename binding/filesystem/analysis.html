H2A Dashboard

Instrumentation dashboard for the H2A protocol (Human-to-Assistant). Monitors friction, resolutions, and epistemic flow across all SOFIA instances. Each friction is a qualified epistemic position — the protocol captures, qualifies, and preserves them as structured data.

Mirror = health diagnostics · Lens = time series · Probe = conformity audit · Right-click any element to navigate.

Trajectory — challenge % iChallenge % per window of N frictions. A descending line = instance going silent. Each window covers a variable time span. Hover for date range.

Friction by marker — timeline iDistribution of the 5 epistemic markers over time. A line going up = more frictions of that type. A flat line at zero = marker never used.

Direction — who contests whom iHorizontal bars: right = persona positions (A), left = orchestrator positions (H). Corroboration = agreement. Contestation = disagreement.

Direction — timeline iEvolution of contestation and corroboration over time. A→H rising = personas push back more. H→A rising = orchestrator challenges more.

Markers by persona iStacked bar: total friction per persona, broken down by marker type. Monochrome = domestication (only sound).

Resolutions by persona iHow frictions are resolved per persona. All ratified = no real arbitration. Mix of ratified/revised/rejected = healthy.

Epistemic flow — H vs A iWho brings substance to the exchange. H >> A = orchestrator drives. A >> H = persona drives. Balanced = co-construction.

Frictions per session — timeline iAverage number of frictions and resolutions per session over time. Rising = more friction per session. Falling = sessions produce less signal.

Directional matrix — computed friction

Direction from initiative + marker. Corroboration = sound. Contestation = contestable, simplification, blind_spot, refuted.

Persona A corroborates H A contests H H corroborates A H contests A Ratio A→H / H→A

Ratio > 2 = persona contests more than contested — healthy friction. Ratio ~ 1 = balanced. Ratio < 0.5 = potential domestication.
